# Ea Writing Format and Signatures

Status: DRAFT  
Priority: Critical

## Purpose

Define standard formatting rules for Ea emails, documents, reports, memos and signatures.

## Required sections

| Section | Description |
|---|---|
| Default email structure | Subject, greeting, body, closing |
| English signature | Full and short versions |
| Norwegian signature | Full and short versions |
| Quote/proposal format | Pricing, lead time, assumptions, validity |
| Technical response format | Product fit, limitations, application notes, caveats |
| Follow-up format | Polite reminder structure |
| Meeting reschedule format | Short, clear scheduling language |
| Date/time format | Europe/Oslo, dd.mm.yyyy where relevant |
| Currency format | NOK/EUR/USD conventions |
| Units | °C, µm, m², g/m², L, kg |
| Attachment wording | How to mention attachments |
| Caveats | Legal, financial, technical and commercial caveats |
