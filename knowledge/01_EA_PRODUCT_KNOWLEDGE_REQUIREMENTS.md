# Ea Product Knowledge Requirements

Status: DRAFT  
Priority: Critical

## Purpose

Define the product knowledge Ea needs to draft accurate emails, technical replies, meeting memos, due diligence notes, product recommendations and proposal support.

## Required product files

Recommended folder:

```text
knowledge/product/
```

Recommended product files:

```text
PRODUCT_HIREC_R.md
PRODUCT_HIREC_PFW9.md
PRODUCT_HIREC_PFS10.md
PRODUCT_SOLAREX_QUARTZ_SIO2.md
PRODUCT_SOLAREX_TITAN_TIO2.md
PRODUCT_SURFACEGUARD_X.md
PRODUCT_SURFACEGUARD_T.md
PRODUCT_GENTOO_4700.md
PRODUCT_ULTRA_EVER_DRY.md
PRODUCT_TEXTILE_COATING_R_AND_D.md
```

## Product file schema

Each product knowledge file must include:

| Field | Requirement |
|---|---|
| Approved product name | Name Ea must use |
| Legacy names | Previous names, aliases, forbidden names |
| Category | Coating type/application family |
| Short approved description | 1-3 sentence customer-safe summary |
| Main use cases | Approved applications |
| Suitable substrates | Supported surfaces/materials |
| Unsuitable substrates | Known limits |
| Key claims | Source-backed claims only |
| Forbidden/unverified claims | Claims Ea must not use |
| Application method | Spray, wipe, HVLP, gravity-feed, etc. |
| Coverage/consumption | m²/L, m²/kg, g/m² or other |
| Drying/curing | Time/temperature/conditions |
| Durability/service life | Approved wording only |
| Safety handling | SDS-level summary only |
| Regulatory/shipping notes | VOC, DG, transport limitations |
| MOQ/sample policy | Approved minimums and sample restrictions |
| Lead time | Current or reference lead time with source date |
| Pricing | Current or reference pricing; PENDING_REVIEW unless approved |
| Source documents | TDS/SDS/SOP/email/report locations |
| Review date | Last review date |
| Status | DRAFT, PENDING_REVIEW, APPROVED or CANONICAL |

## Naming rules

- Hirec100 -> Hirec-R
- Hirec 450 water-based -> Hirec PFW9
- Hirec 450 solvent-based -> Hirec PFS10
- SiO2 -> SiO₂
- TiO2 -> TiO₂

## Approval rule

New product performance claims, warranty language, durability claims, pricing, regulatory claims and customer-facing technical promises are PENDING_REVIEW unless explicitly approved.
