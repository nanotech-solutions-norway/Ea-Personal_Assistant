# Ea Document Template Requirements

Status: DRAFT  
Priority: Critical

## Purpose

Define the document templates Ea needs for repeatable business output.

## Required template folder

```text
knowledge/templates/
```

## Required templates

| Template | Purpose |
|---|---|
| TEMPLATE_EMAIL_REPLY.md | Standard email reply |
| TEMPLATE_MEETING_MEMO.md | Meeting preparation memo |
| TEMPLATE_MEETING_MINUTES.md | Post-meeting minutes |
| TEMPLATE_FOLLOW_UP_EMAIL.md | Follow-up after meeting/email |
| TEMPLATE_DUE_DILIGENCE_REPORT.md | Company review |
| TEMPLATE_LEGAL_REVIEW_MEMO.md | NDA/contract business-risk review |
| TEMPLATE_CONTRACT_RISK_TABLE.md | Clause/risk table |
| TEMPLATE_FINANCIAL_REVIEW.md | Budget/cost/margin review |
| TEMPLATE_TAX_NOTE.md | Preliminary tax/VAT support note |
| TEMPLATE_PRODUCT_RECOMMENDATION.md | Product selection memo |
| TEMPLATE_TECHNICAL_RESPONSE.md | Technical customer response |
| TEMPLATE_QUOTATION_EMAIL.md | Quote/proposal email |
| TEMPLATE_LEAD_LIST.md | Leads/prospect tracking |
| TEMPLATE_ACTION_REGISTER.md | Action tracking |
| TEMPLATE_SESSION_CLOSE.md | End-of-session handoff |

## Required sections per template

- Purpose.
- When to use.
- Required inputs.
- Output structure.
- Approval requirements.
- Confidentiality classification.
- Source/evidence section.
- Review status.
- Version/date.

## Professional-boundary rule

Legal, financial, accounting and tax templates are business-support templates only and must not claim to replace qualified advisors.
