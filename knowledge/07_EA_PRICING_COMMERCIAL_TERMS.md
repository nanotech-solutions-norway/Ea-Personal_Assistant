# Ea Pricing and Commercial Terms

Status: DRAFT  
Priority: Critical

## Purpose

Define the commercial knowledge Ea needs for quotes, proposals, emails, supplier/customer negotiations and meeting preparation.

## Required fields

| Field | Description |
|---|---|
| Product/service | Item being priced |
| Price | Current/reference price |
| Currency | NOK, EUR, USD or other |
| Unit | kg, L, m², project, hour, etc. |
| MOQ | Minimum order quantity |
| Sample policy | Plates, liquid samples, test kits, etc. |
| Lead time | Typical lead time with source date |
| Incoterms | FCA, EXW, DAP or other |
| Payment terms | Payment split, L/C, due dates |
| Discount rules | Approved/case-specific discount logic |
| Warranty/service life | Approved wording only |
| Validity | Quote validity period |
| Approval requirement | When explicit approval is mandatory |
| Source | Email, quote, supplier document, report |
| Status | DRAFT, PENDING_REVIEW, APPROVED or CANONICAL |

## High-risk rule

Pricing, discounts, warranties, delivery commitments, exclusivity, service-life claims and payment terms are PENDING_REVIEW unless explicitly approved.

## Use rule

Ea may prepare draft commercial language, but must not make final pricing, warranty, delivery or exclusivity commitments without approval.
